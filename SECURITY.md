# Security Policy

## Reporting

Report security vulnerabilities via GitHub Security Advisories:
https://github.com/hop-top/fit/security/advisories/new

Do **not** file public issues for security bugs.

## Response SLA

- Acknowledgment: 48 hours
- Initial assessment: 5 business days
- Patch: depends on severity

| Severity | Timeline |
|----------|----------|
| Critical | 24 hours |
| High | 72 hours |
| Medium | 7 days |
| Low | Next release |

## Scope

- All published packages (fit, @hop/fit, fit-py, fit-rs, fit-php)
- Spec files
- Build infrastructure

## Out of scope

- Frontier LLM provider vulnerabilities (report to provider)
- Advisor model training issues (see fit-coach)
